# cleanup.sh

`cleanup.sh` is a macOS shell script designed to automate file cleanup and organization. It creates destination directories when needed, previews detected files, and sorts them into the appropriate folders. The script also includes a dry-run mode, allowing safe testing before applying any changes.

## Features

- Automated file organization
- Automatic folder creation
- File preview before execution
- Dry-run support for safe validation
- Lightweight and easy to use

## Usage

```bash
chmod +x cleanup.sh
./cleanup.sh
```
